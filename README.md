# :computer: Trakker
</br>

## :page_facing_up: 프로젝트 개요

다양한 여행 플래너를 사용하면서 다른 유저들과 여행 계획을 공유 하고 싶었습니다.

이에 착안하여 KAKAO Map API, 한국관광공사 API, Spring Framework를 이용해 국내 여행 플래너 공유 웹 애플리케이션을 제작했습니다.

</br>
